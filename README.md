# markdown → pdf

A minimal browser-based Markdown editor with one-click PDF export. No uploads, no servers — everything runs locally.

## Features

- Live split-view preview
- Drag & drop `.md` / `.txt` / `.markdown` files
- Clean, print-optimized PDF export (A4)

## Usage

```sh
bun install
bun run dev
```

Open the app, write Markdown, click **Export PDF**.
